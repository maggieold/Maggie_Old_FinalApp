Build Your Own Design!

Description:

This site includes a generative web application that allows the user to create circles that are controlled by a graphical user interface. Use the controls see what you can make of it!

What you can control:

- How many circles are in each set
- What size each set of circles can grow or shrink to be
-How thick or thin the weight of each circle will be
- And you have the option to randomize the sizing for a little more of a random design look.
AND you can click and drag to draw with your own circles!
